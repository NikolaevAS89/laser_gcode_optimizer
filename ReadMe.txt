Программа может оптимизировать путь для резки лазерным гравером, однако,
нет варианта выбора пути с наименьшим отклонением от траектории движения.
Хороший результат может быть получен и без оптимизации пути путем более
правильного построения рисунка в редакторе. Главное не допускать разрывов в
линиях.
